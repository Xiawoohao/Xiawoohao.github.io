錄音逐字稿接案 (2025/07/25)

Privioisly on

在Pro360接到人生中第一個案子，工作內容是聽音檔並且轉成逐字稿。當時報價3500，音檔時長三個多小時，總共有三四個人在講話。並且要三天內交稿。

Hugeee Problem

原本想說三小時音檔不就三四小時完成嗎，哇靠一個小時賺一千！？！但現實往往是骨感的…，我暴打了三十分鐘手開始酸了，但音檔只跑了五分鐘。我勒個豆豆，根本不可能在短時間完成。我心灰意冷的上頂樓整根，想說查一下有沒有什麼AI工具可以幫忙（因為之前有查過音檔轉逐字稿，很多網站都要錢…）

Whisper AI Tools

Then I find a tool calls Whisper 它可以將音檔轉成文字，並且支援各種主要語言，甚至還可以做翻譯。
接下來就是基本的Set up

流程

1.因為它需要比較好的GPU跑會比較快，我使用colab。它可以選用
2.右上角按connect就會連接到colab提供的環境
3.下這兩個指令後就可以開始轉換了
pip install -U openai-whisper
sudo apt update && sudo apt install ffmpeg
4.將你要轉換的音檔放入左邊的files區域，並且下
whisper [你的音檔名] --model [model]
※我是用medium or turbo
5.接下來它會輸出成五種格式檔：txt、tsv、srt、vtt、json

總結

它目前的錯誤率大概10%左右，並且他的斷句是不會偵測人聲來判斷，所以不管是誰講只要有停頓都算是斷句。所以我還需要在重新排版並且將誰的話標記。但這還是解決了我非常大的問題，感恩whisper!

當然也可以用本機的方式去使用whisper，可以到他們的github去看教學，或是上網查會有非常多資源能夠幫助你！
